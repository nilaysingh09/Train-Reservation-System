🚆 Trans-City Railway Online Platform (TCROP)

A simple Train Reservation System GUI application built using Python. This project simulates a railway booking platform with user authentication and database integration using MySQL.

📌 Features
🔐 User Login System
📝 New User Registration
🔁 Password Confirmation Validation
🗄️ MySQL Database Integration
⚠️ Error Handling for invalid inputs
🖥️ GUI-based interaction using EasyGUI
❌ Exit / Cancel options
🛠️ Tech Stack
Language: Python 🐍
Libraries Used:
pandas
easygui
os
mysql-connector-python (or pymysql, mention what you used)
Database: MySQL 🗄️
